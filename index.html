My API

API er et sett med regler og mekanismer som gjør at én applikasjon eller komponent kan samhandle med andre. API-et er en programmatisk måte for forskjellige programvarekomponenter å kommunisere med hverandre. Det er et sett med regler som lar én programvareapplikasjon snakke med en annen. Det er en programvareformidler som gjør at to applikasjoner kan snakke med hverandre. Med andre ord, et API er budbringeren som leverer forespørselen din til leverandøren du ber om og deretter leverer svaret tilbake til deg.
